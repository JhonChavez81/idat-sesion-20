¿Qué es Bootstrap?

Bootstrap es el framework front-end más popular del mundo para diseñar sitios web responsivos y dirigidos a dispositivos móviles.

Proporciona una amplia colección de fragmentos de código HTML, CSS y JavaScript reutilizables para construir rápidamente interfaces web modernas sin partir de cero.

  • Filosofía Mobile-First estricta.
  • Consistencia visual en todos los navegadores.
  • Variables CSS y personalización con Sass.
  • Amplia comunidad y documentación robusta.

El Sistema de Rejilla

Contenedores (.container)

El bloque de construcción base. Centran horizontalmente el contenido del sitio web, definiendo el ancho máximo en distintos puntos de quiebre responsivos.

Filas (.row)

Envoltorios esenciales para las columnas. Aseguran que estas se alineen correctamente y manejan márgenes negativos para un flujo perfecto en pantalla.

Columnas (.col)

Definen el ancho del contenido. La rejilla se basa en 12 columnas por fila, permitiendo layouts ultra complejos mediante clases como .col-md-6.

Botones (.btn)

Diseñados para acciones rápidas con soporte para múltiples tamaños y estados. Alto contraste garantizado.

<button class="btn btn-dark">
  Acción Principal
</button>

Alertas (.alert)

Proporcionan retroalimentación contextual clave para acciones del usuario con estilos definidos.

<div class="alert alert-light">
  Proceso exitoso.
</div>
Proceso exitoso completado.

Formularios Modernos

  • Controles Consistentes: .form-control unifica el diseño en todos los navegadores.
  • Input Groups: Extienden controles añadiendo texto o botones laterales.
  • Validación Integrada: Estilos visuales robustos basados en HTML5.
  • Switches modernos: Casillas de verificación estilo interruptor.

Uso de Componentes

  • Grid y Layout (35%)
  • Tarjetas y Botones (30%)
  • Formularios (20%)
  • Navegación y Modales (15%)

El Grid System es la base indispensable de la estructura. Componentes como tarjetas y botones forman el núcleo visual de las interacciones diarias.
